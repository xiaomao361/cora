# Vision alignment

Cora's reusable center is a small fact-and-decision loop:

1. collect bounded error evidence;
2. aggregate it into stable Problems;
3. apply explicit, versioned product-line judgment;
4. let Agents investigate and record outcomes;
5. use durable Cases to improve a private Pack through review.

The implementation stays aligned when the public Core remains generic, Packs
remain private and auditable, facts are never silently rewritten by inference,
and automatic learning does not bypass human review. UI, broad observability,
and autonomous remediation are later surfaces, not prerequisites for this loop.
